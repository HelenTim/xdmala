## 51、我想把这样格式「1）我是一个点」的字符串改成为高亮状态的？

```
1）我是一个点 -> <mark>1）我是一个点</mark>
```

做法：

```
(^\d+）[\s\S]*) -> 正则查找
<mark>$1</mark> -> 替换
```

> 思路：开头是数字的（1位或多于1位），匹配一个「）」，任意字符（0位 or 多位），整体加括号，用于在替换时，使用「$1」表示这个要替换的字符串，说白了，我用圆括号就是为了让匹配字符串加上前缀和后缀
